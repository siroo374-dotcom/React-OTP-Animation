# React OTP Animation

A modern and premium OTP Verification UI built with React and Vite.

## Features

- Modern glassmorphism UI
- Smooth OTP input animations
- Interactive typing effects
- Responsive design
- Clean and reusable code
- Built with Framer Motion
- Lucide React icons
- Fast Vite development environment

## Technologies

- React
- Vite
- Framer Motion
- Lucide React
- CSS3

## Installation

```bash
npm install
```

Run the project:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
